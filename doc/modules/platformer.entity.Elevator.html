Module platformer.entity.Elevator

A custom dynamic phyzzle.Entity which moves up and down and lifts other dynamic entities.

Extends phyzzle.Entity.

Functions

init ()
Initialize the Elevator.

Records the starting position.

Called automatically by Entity:new.

collision (e2, dx, dy)
Override the phyzzle.Entity:collision function.

If moving upward checks and the other Entity is dynamic phyzzle.Entity.static = false then move the entity else touch the entity.

Parameters:

  • e2 The other entity.
  • dx Change in x.
  • dy Change in y.
update (dt)
Override the Entity update function.

Parameters:

  • dt if traveled past the distance change direction
