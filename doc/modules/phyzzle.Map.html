Functions

new (t)
Create a new map.

-- w,h,zs can be overridden.

Parameters:

  • t Optional table of overrides and initial state.

Returns:

    Map instance.
init ()
Initialize the map by creating the zones.

Called automatically by Map:new.

Override in child maps to specify custom init functionality. All init functions will always be called unlike other overriden functions.

add (e)
Add an entity to the map.

Parameters:

  • e The Entity to add.

Returns:

    The added Entity.
get (t)
Get a list of entities that intersect a AABB.

Useful for checking for collisions in specific places.

Every Entity is an AABB. A generic AABB needs an x,y,w,h.

Parameters:

  • t AABB to check for collisions.

Returns:

    table Table of entities indexed by Entity table key.
move (e, dx, dy)
Move an entity in the map and check for collisions.

Will either trigger the entity's move or collision response handler. Movement is done one axis at a time, X first, Y second.

Parameters:

  • e The Entity to move in the map.
  • dx The amount to move in the X axis.
  • dy The amount to move in the Y axis.
zone (e)
Update Entity.zones base on the new location.

Called automatically by Map:add. Needs to be called after manually setting an Entity locaiton x or y. You only have to manually set an Entity location if you want to avoid collision checks.

Parameters:

  • e The `Entity to zone.
update (dt)
Update all dynamic entities.

If an entity is flagged for removal it is removed.

Parameters:

  • dt Change in time or delta time.
region (t)
Get a zone region that encompasses the supplied AABB. The AABB needs an x,y,w,h.

Useful for creating viewport based renderers or querying sections of the map.

Parameters:

  • t AABB zones need to cover.

Returns:

  1. x1 The top-left zone's x location.
  2. y1 The top-left zone's y location.
  3. x1 The bottom-right zone's x location.
  4. y1 The bottom-right zone's y location.

Fields

w
Map width in pixels.
h
Map height in pixels.
zs
The zone size, the width and height of a zone in pixels.

A map will be divided in to a grid of zones to optimise collisions and renderering.

entitiesCount
Total number of entities in the map.
